## 📝 重构时的注意事项

### ✅ 必须做到：
1. **完全复制所有 UI 元素**
2. **保持相同的样式和布局**
3. **保持相同的交互效果**
4. **保持相同的动画**
5. **保持相同的响应式行为**

### ⚠️ 可以改变的：
1. ✅ 代码结构（组件拆分）
2. ✅ 状态管理方式（Zustand）
3. ✅ 文件组织（多文件）
4. ✅ 类型定义（TypeScript）
5. ✅ API 调用方式（统一 API 层）

### ❌ 不能改变的：
1. ❌ UI 外观
2. ❌ 用户交互流程
3. ❌ 视觉效果
4. ❌ 布局结构
5. ❌ 颜色和字体
